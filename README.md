# Myco-Naturals: Smart Cordyceps Farming Ecosystem

A premium, state-of-the-art platform designed to revolutionize Cordyceps militaris cultivation, franchise management, and direct-to-consumer marketplace. This system empowers rural farmers with AI-driven guidance, real-time monitoring visualizations, and a guaranteed market ecosystem.

> [!IMPORTANT]
> **Status**: Current deployment (GitHub Pages) is **Frontend-Only**. The backend services (Node.js/Express) are currently in development and the live version uses local storage and mock integrations for demonstration purposes.

🚀 **[View Live Site](https://edwinnova-2026.github.io/Myco-Naturals/)**

---

## 🛠️ Technology Stack (By Component)

The system is built using modern, high-performance web technologies, prioritizing a seamless user experience and accessibility for rural farmers.

### 1. **Core Frontend Foundation**
- **HTML5 & JSX**: Structural logic and semantic markup for SEO and accessibility.
- **CSS3 (Vanilla & Tailwind 4)**: Premium aesthetics featuring glassmorphism, smooth gradients, and a sleek dark-mode-first design.
- **JavaScript (ES6+)**: Core application logic and dynamic user interactions.
- **React 19**: The primary framework for building modular, high-performance UI components.

### 2. **Intelligence & Automation**
- **OpenAI API**: Powers the **Smart Cultivation Assistant**, providing real-time, context-aware answers to farming queries.
- **i18next**: Comprehensive multilingual support for **English, Tamil, Hindi, Kannada, and Telugu**, ensuring farmers can use the platform in their native language.

### 3. **Data Visualization & Animations**
- **Recharts**: High-performance charting library used in the Farmer Dashboard to visualize sensor data (Temperature, Humidity, CO2).
- **Framer Motion**: Smooth, premium micro-animations and page transitions to enhance engagement.

### 4. **Infrastructure & Utilities**
- **Vite 8**: Next-generation frontend tooling for blistering fast builds and hot module replacement.
- **React Router 7**: Managed navigation and dynamic routing across the ecosystem.
- **EmailJS & WhatsApp API**: Direct communication channels for franchise onboarding and order processing.
- **Lucide-React & React Icons**: Modern iconography for intuitive navigation.

## 📦 What We Built (System Components)

### 🌾 **Farmer Dashboard (The Cultivation Hub)**
- **Real-time Sensor Monitoring**: Visualized graphs tracking critical cultivation parameters.
- **Batch Management**: Tools to track growth stages from inoculation to harvest.
- **Growth Projections**: Estimated yields and timelines based on current batch data.

### 💼 **Franchise & Training Ecosystem**
- **4-Day Training Program**: Detailed curriculum (Day 1-4) covering everything from sterile handling to market strategies.
- **Verified Buy-back Guarantee**: Integrated assurance for farmers that their products have a guaranteed buyer.
- **Training Kit Booking**: Streamlined process to join the franchise community for ₹7,000.

### 🛒 **Consumer & Farmer Marketplace**
- **Premium Shop**: Curated list of Cordyceps products (Tea blends, Dried Fruit Bodies).
- **Dynamic Cart Logic**: Intelligent pricing engine that adjusts costs based on volume and variants.
- **Buy-back Integration**: A dedicated portal where franchise farmers can sell their grain-based Cordyceps back to the admin.

### 🤖 **Smart AI Assistant**
- A specialized chatbox trained on Cordyceps cultivation data.
- Capable of guiding users through dosage, cultivation troubleshooting, and franchise benefits.

## 🚀 Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/edwinnova-2026/Myco-Naturals.git
   ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run the Application**:
   ```bash
   npm run dev
   ```
   *The application will launch on `http://localhost:5173`.*
