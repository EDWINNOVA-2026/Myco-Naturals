# Myco-Naturals: Smart Cordyceps Farming Ecosystem

A premium, full-stack platform for Cordyceps militaris cultivation, franchise management, and direct-to-consumer marketplace. This platform empowers farmers with AI-driven guidance, real-time monitoring, and a guaranteed market through a buy-back ecosystem.

🚀 **[View Live Site](https://edwinnova-2026.github.io/Myco-Naturals/)**

---

## 🛠️ Technology Stack

| Layer | Technologies |
| :--- | :--- |
| **Frontend** | React 19, Vite 8, Tailwind CSS 4, Framer Motion |
| **Backend** | Node.js, Express 5, MongoDB (via Mongoose) |
| **Languages** | JavaScript (ES6+), JSX, HTML5, CSS3 |
| **State & Logic** | React Hooks, Context API |
| **Integrations** | WhatsApp API, EmailJS, OpenAI API (AI Assistant) |
| **Internationalization** | i18next (English, Tamil, Kannada, Hindi, Telugu) |

## ✨ Key Features

### 1. 💼 Franchise & Training
- **4-Day Offline Training**: Intensive in-person program covering market fundamentals to harvesting.
- **₹7,000 Complete Package**: All-inclusive training fee with certificates and materials.
- **WhatsApp Integration**: Applicant credentials and inquiries are routed directly to admin via WhatsApp.
- **Buy-back Guarantee**: Assured market for all cultivated Cordyceps.

### 2. 🛒 Consumer Marketplace
- **Smart Cart Logic**: Dynamic pricing based on quantity and product variants.
- **Premium Products**: Range of Cordyceps tea blends, dried mushrooms, and supplements.
- **Secure Ordering**: Streamlined checkout process with direct communication.

### 3. 👨‍🌾 Farmer Dashboard
- **Real-time Monitoring**: Visualized sensor data for temperature, humidity, and CO2.
- **Batch Tracking**: Manage multiple cultivation cycles simultaneously.
- **AI Cultivation Assistant**: Voice-enabled AI to guide farmers through complex steps.

### 4. 🌍 Multilingual Accessibility
- Full platform support for multiple Indian languages to ensure inclusivity for rural farmers.
- Localized UI for all major states: TN, KA, HI, TE.

## 🚀 How to Run Locally

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### 1. Clone the repository
```bash
git clone https://github.com/edwinnova-2026/Myco-Naturals.git
cd Myco-Naturals
```

### 2. Setup Frontend
```bash
npm install
npm run dev
```
The site will be available at `http://localhost:5173`.

### 3. Setup Backend (Optional for static dev)
```bash
cd server
npm install
node index.js
```
The server will run on `http://localhost:5000`.
